# findln — V2 Specification

V2 introduces AI-powered features, better categorization, and seller guidance.

## Image Guidance

- Users get guidance on how to take good photos of their items
- Tips shown during the upload flow (e.g. good lighting, plain background, show details)

## AI Descriptions

- Descriptions are auto-generated via AI from the uploaded photo
- User can edit the generated description before posting

## Categories

- A more sophisticated category model beyond the basic V1 filters
- Possible categories: clothing by type (jackets, pants, shoes), sports by type, age ranges, seasons

## User Onboarding

- Guidelines for sellers shown during first item post or in a dedicated help section
- Examples:
  - "Wasche deine Kleidung vor dem Verkauf"
  - "Verkaufe keine Kleidung mit Löchern"
  - "Verkaufe keine kaputten Sachen"
